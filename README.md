# 🌍 Climate Change & Environmental Impact Analysis Dashboard (Power BI)

## Project Overview
This project is an interactive **Climate Change Analytics Dashboard** developed using Microsoft Power BI.  
The dashboard analyzes environmental indicators such as **temperature trends, CO₂ emissions, renewable energy usage, sea level rise, rainfall patterns, and extreme weather events** across different countries and years.
The goal of this project is to transform complex climate datasets into **interactive visual insights** that help understand global environmental trends and support data-driven analysis.
## Objectives
- Analyze global **temperature trends over time**
- Identify **countries with high CO₂ emissions**
- Study **renewable energy adoption across countries**
- Understand environmental indicators such as **sea level rise and rainfall patterns**
- Provide **interactive visualizations** for climate data exploration
